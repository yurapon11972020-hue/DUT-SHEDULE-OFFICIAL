DUT SaaS не запускается через GitHub Pages

Этот проект использует Node.js backend, API, авторизацию, JSON-базу и скрытую админку /admin/. GitHub Pages отдаёт только статические HTML/CSS/JS файлы, поэтому полная версия должна быть опубликована на Render, Railway, Fly.io или другом Node.js-хостинге.

Код хранится на GitHub, а рабочая ссылка должна вести на публичный Node.js сервер.
