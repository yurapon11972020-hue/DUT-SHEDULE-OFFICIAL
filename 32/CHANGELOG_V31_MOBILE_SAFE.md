# DUT SaaS v31 — mobile safe mode

Что исправлено:

- Модальное окно создания/редактирования задачи теперь нормально прокручивается на телефоне.
- В Liquid Glass больше не ломается прокрутка формы: у `.modal` принудительно включён `overflow-y: auto`.
- Кнопки «Отмена» и «Сохранить» на телефоне закреплены внизу модального окна и остаются доступными.
- Поля дат на мобильных и планшетных экранах идут друг под другом, чтобы не сжиматься.
- Палитра цветов задачи на телефоне стала компактной: карточки цветов меньше и не раздувают форму.
- На телефонах включён строгий mobile-safe режим: отключены тяжёлые blur/backdrop-filter, псевдоблики, фоновые орбы, анимации и лишние hover-переходы.
- Drag-and-drop карточек на телефоне отключён, чтобы браузер не пытался запускать тяжёлый перетаскиваемый режим. Для телефона остаются кнопки «Назад» и «Дальше».
- Добавлена блокировка фоновой страницы при открытой модалке, чтобы скролл не конфликтовал с прокруткой формы.

Как проверить:

1. Залей файлы в GitHub в ту же папку, из которой Render собирает проект.
2. Сделай deploy latest commit на Render.
3. Открой сайт на телефоне.
4. Войди в аккаунт с тарифом Max.
5. Открой доску и нажми «+ Добавить задачу».
6. Проверь, что форма листается вверх-вниз, а кнопка «Сохранить» доступна.
7. Проверь Liquid Glass: на телефоне он будет легче, а на компьютере останется красивым.
