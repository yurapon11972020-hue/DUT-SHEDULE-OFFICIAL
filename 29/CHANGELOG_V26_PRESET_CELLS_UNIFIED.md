# DUT SaaS v26 — единый стиль ячеек пресетов

Что изменено:

1. Все ячейки пресетов в настройках дизайна теперь выглядят в одном стиле.
2. У пресета Liquid Glass убрана отдельная декоративная подсветка, из-за которой он визуально выбивался из сетки.
3. Сам пресет Liquid Glass не удалён: он по-прежнему применяет жидкое стекло к интерфейсу после нажатия.
4. Описание Liquid Glass укорочено, чтобы карточка не казалась тяжелее остальных.
5. В режиме Liquid Glass сетка пресетов больше не получает отдельные блики и псевдоэлементы только для одной карточки.
6. Сохранены предыдущие правки:
   - чистая главная без лишних UX-блоков;
   - скрытый /quality;
   - убран бейдж «Текущий: ...» в тарифах;
   - цены по языкам и валютам;
   - Liquid Glass как материал интерфейса.

Проверка:

```bash
node --check public/app.js
node --check server.js
```

Ожидаемый результат:

- открыть профиль;
- перейти в Настройки → Дизайн;
- посмотреть блок «Готовые стили»;
- все ячейки пресетов должны быть одинакового типа;
- Liquid Glass должен выделяться только названием, а не отдельной карточной графикой;
- при нажатии Liquid Glass сам стиль сайта всё равно должен применяться.
