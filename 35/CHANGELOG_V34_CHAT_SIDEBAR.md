# DUT-SCHEDULE-TASK v34 — боковая выдвижная панель

## Что изменено
- Верхняя навигация перенесена в боковую панель.
- Панель открывается слева по кнопке-гамбургеру и выдвигается как sidebar в ChatGPT.
- Добавлен затемняющий overlay.
- Закрытие работает по крестику, клику по фону, переходу по ссылке и клавише Esc.
- На мобильной версии sidebar облегчен: отключены тяжелые blur-эффекты, чтобы не создавать лаги.
- Верхняя панель стала чище: логотип + кнопка открытия меню.

## Как проверить
1. Запустить проект: `npm start`.
2. Открыть главную страницу.
3. Нажать кнопку меню слева от логотипа.
4. Проверить, что панель выезжает сбоку.
5. Перейти по пунктам меню — панель должна закрываться автоматически.
6. Проверить то же самое на телефоне.
