## Как проверить

1. Запустить проект:

```bash
npm start
```

2. Открыть сайт:

```text
http://localhost:3000
```

3. Создать обычного пользователя через регистрацию.

4. Вручную открыть админку:

```text
http://localhost:3000/admin/
```

5. Войти:

```text
email: yurapon11972020@gmail.com
password: 1488675242
```

6. Найти обычного пользователя через поиск.

7. Проверить действия:

- выдать Pro;
- выдать Max;
- деактивировать;
- попробовать войти обычным пользователем;
- активировать обратно;
- сбросить доску;
- удалить пользователя.

## Важно для настоящего деплоя

Для реального публичного сайта пароль лучше вынести в переменные окружения:

```text
ADMIN_EMAIL=...
ADMIN_PASSWORD=...
COOKIE_SECRET=...
```

В учебной версии значения уже заданы по умолчанию, чтобы проект быстро запускался и проверялся.
