# DUT SaaS v37 — bugfix support and drag

- Исправлено пустое состояние колонки после перетаскивания задачи: если задача уже есть в колонке, блок "Здесь пока нет задач" удаляется сразу.
- Счётчики задач в колонках обновляются сразу после drag-and-drop без ожидания полного перерендера.
- Убрана лишняя информационная карточка "Навигация" из боковой панели.
- Мини-чат поддержки теперь обновляет только область сообщений при автопроверке, поэтому набранный пользователем текст не пропадает.
- Автообновление поддержки ускорено: открытый чат проверяет новые сообщения чаще, закрытый чат обновляет счётчик непрочитанных.
- Сохранены предыдущие исправления: единый логотип в админке, упрощение подсказок, поддержка ответов администратора.
